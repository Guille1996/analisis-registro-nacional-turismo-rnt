## Uso de GitHub Copilot

Durante el desarrollo del proyecto se utilizó GitHub Copilot en Visual Studio Code como asistente de programación. Copilot apoyó la revisión de la estructura del repositorio, la explicación de fragmentos de código, la generación de sugerencias para mejorar funciones de limpieza de datos y la corrección de errores durante la ejecución del notebook.

### Prompts usados en Copilot

- "Revisa la estructura de este proyecto de análisis reproducible en Python."
- "Ayúdame a corregir este error de pandas."
- "Mejora este gráfico de barras horizontal con matplotlib."
- "Ayúdame a redactar conclusiones descriptivas sin afirmar causalidad."

### Qué se revisó o ajustó

Las sugerencias de Copilot fueron revisadas antes de incorporarlas al proyecto. Se ajustaron nombres de columnas, rutas de archivos, títulos de gráficos y explicaciones del notebook de acuerdo con la estructura real del dataset.

### Reflexión crítica

Copilot fue útil para acelerar la escritura y revisión del código, pero las decisiones finales sobre la pregunta de análisis, la interpretación de resultados y las conclusiones fueron revisadas por el autor del proyecto.
